# Micafila Portfolio – Refinement Suggestions (Owner View)

Target audience: Non-tech-savvy industrial users. Goal is to refine, not redesign.

---

## 1. Active Tab Indicator
- Current: Red underline is thin and easy to miss
- Fix: Increase thickness to 3–4px, make the active tab label bold

---

## 2. Profile Photo
- Current: Square crop feels unfinished
- Fix: Add subtle rounded corners or a thin border to make it look intentional

---

## 3. Name & Title Spacing
- Current: Name and role title are too close together
- Fix: Add more vertical spacing between them for better readability, especially for older users

---

## 4. Add Work Card
- Current: Grey placeholder box blends into the background, looks like a broken image
- Fix: Add a visible border or a light teal/blue tint to make it clearly an action button

---

## 5. Achievement Section
- Current: "Achievement" and "Galaxy Awards" are buried below the fold with no visual distinction
- Fix: Add a small gold/yellow badge icon to make it feel earned and more visible

---

## 6. Email Display
- Current: Full email shown publicly with no icon or masking
- Fix: Add a small email icon for context; consider partial masking for privacy

---

## 7. Font Refinements
No font change needed — just apply size, weight, and color contrast to existing font.

| Element         | Issue                        | Fix                                  |
|-----------------|------------------------------|--------------------------------------|
| Name            | Medium size, plain weight    | Larger + bold                        |
| Role/Title      | Same size as name            | Slightly smaller + accent color      |
| Location/Email  | Same tone as role            | Smaller + light grey                 |
| Tab labels      | All same weight              | Active tab bold, inactive regular    |
| Card titles     | Not visible                  | Small bold label on hover            |

### Why this matters for industrial users:
- They may be reading on a tablet or in poor lighting
- Strong size and weight contrast reduces cognitive load
- Color-coded hierarchy (name vs role vs metadata) helps scan quickly

---

## Summary
Six layout fixes + one font pass. No structural changes. The existing layout works well for this audience — these are polish-only refinements.
